# ikamemo

Splatoon 3のバトルメモ生成ツールです。テンプレートから試合結果のメモファイルを生成します。

## 使い方

### 基本的な使い方

```bash
go run . -w "プラFR" -r エリア -s オヒョウ -o win
```

### オプション

| フラグ | 省略形 | 説明 | 例 |
| -------- | -------- | ------ | ----- |
| `--weapon` | `-w` | 使用武器名 | `-w "プラFR"` |
| `--rule` | `-r` | ルール（部分一致対応） | `-r エリア` `-r ヤグ` |
| `--stage` | `-s` | ステージ名（部分一致対応） | `-s オヒョウ` `-s 空港` |
| `--outcome` | `-o` | 結果（win/lose） | `-o win` |
